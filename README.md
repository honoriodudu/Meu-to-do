# Welcome to your Dyad app

Este é um projeto web moderno construído com React, TypeScript, Tailwind CSS e shadcn/ui.

## Tecnologias Utilizadas

- **React** - Biblioteca principal para interface
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização moderna
- **shadcn/ui** - Componentes UI acessíveis
- **React Router** - Navegação entre páginas

## Páginas Disponíveis

- **Início** - Página principal com informações do projeto
- **Sobre** - Informações sobre o projeto e tecnologias
- **Serviços** - Serviços oferecidos pela aplicação
- **Contato** - Formulário de contato
- **Preços** - Planos e preços dos serviços
- **404** - Página não encontrada

## Estrutura do Projeto

```
src/
├── components/          # Componentes reutilizáveis
├── pages/              # Páginas da aplicação
├── lib/                # Utilidades e configurações
├── hooks/              # Custom hooks
├── utils/              # Funções utilitárias
└── globals.css         # Estilos globais
```

## Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra o navegador em `http://localhost:3000`

## Personalização

Você pode personalizar cores, fontes e estilos nos arquivos:
- `tailwind.config.ts` - Configuração do Tailwind
- `src/globals.css` - Estilos globais
- `src/components/` - Personalizar componentes existentes

## Deploys

O projeto está configurado para deploy fácil em plataformas como Vercel, Netlify, etc.
